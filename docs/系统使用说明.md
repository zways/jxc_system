## 一、系统简介

进销存管理系统是一套面向企业的**多企业**进销存与财务管理软件，支持：

- **采购管理**：供应商、采购订单、收货、入库记录
- **销售管理**：客户、销售订单、发货、出库、退货与换货
- **库存管理**：实时库存、调拨、调整、盘点
- **财务管理**：应收/应付、收支明细、对账
- **报表中心**：销售/采购/库存/财务报表与导出
- **系统设置**：部门、仓库、门店、角色权限、用户、商品分类与单位、业务员等

技术栈：前端 Vue 3 + Element Plus，后端 Laravel（PHP），数据库 MySQL，缓存/会话/队列 Redis。
