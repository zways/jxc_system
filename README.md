# 进销存管理系统

## 项目概述

这是一个功能全面的进销存管理系统，支持批发销售、零售销售、供应商往来帐务管理、客户往来帐务管理、销售换货、业务员管理、POS 端销售管理、财务管理、库存盘点及连锁店管理等功能。

## 系统特点

-   **完整功能覆盖**: 包含采购、销售、库存、财务等所有进销存环节
-   **现代化架构**: 基于 Laravel 框架和 RESTful API 设计
-   **可扩展性**: 模块化设计，易于扩展新功能
-   **数据安全**: 完整的权限管理和数据验证
-   **高效性能**: 优化的数据库设计和查询

## 技术栈

-   **后端**: Laravel 12.x / PHP 8.x
-   **数据库**: MySQL / SQLite
-   **API**: RESTful API 设计
-   **缓存**: Redis (可选)

## 数据库结构

系统包含以下主要数据表：

-   `products` - 产品表
-   `product_categories` - 产品分类表
-   `suppliers` - 供应商表
-   `customers` - 客户表
-   `purchase_orders` - 采购订单表
-   `sales_orders` - 销售订单表
-   `warehouses` - 仓库表
-   `inventory_transactions` - 库存流水表
-   `inventory_adjustments` - 库存调整表
-   `inventory_counts` - 盘点表
-   `accounts_receivable` - 应收账款表
-   `accounts_payable` - 应付账款表
-   `financial_transactions` - 财务交易表

## 安装说明

1. 克隆项目代码
2. 安装依赖包
    ```
    composer install
    ```
3. 配置环境变量
    ```
    cp .env.example .env
    php artisan key:generate
    ```
4. 配置数据库连接
5. 运行数据库迁移
    ```
    php artisan migrate
    ```
6. 填充示例数据
    ```
    php artisan db:seed
    ```
